Mots Glissés.

Intro

Ce projet a été développé par Alexandre COURRIEU et Adrien SCAZZOLA dans le cadre du cours de POO de 2023-2024 de l'ESILV

Principe du jeu

Le but du jeu est de trouver un maximum de mots français dans une matrice, à partir d'une grille affichée sur la console. Si vous prenez trop de temps à répondre ou que le mot que vous avez choisi n'est pas français, ou qu'il n'existe pas sur le plateau, vous passez votre tour.

Comment démarrer?

Le jeu se joue en console, il suffit de lancer le programme et de suivre les instructions. Il est préférable d'afficher la console en plein écran avant de commencer.

Tout d'abord, il vous faudra saisir les noms des joueurs. Par défauts, ils sont définis à Player1 et Player2.

Ensuite vous devez sélectionner le temps maximal de la partie. Il est défini en secondes par un sélecteur. Utilisez les touches Up and Down pour choisir la valeur qui convient.

Vous pouvez ensuite sélectionner le temps que chaque joueur a pour trouver un mot, par les mêmes moyens que précédemment.

Vous arriverez ensuite sur le menu principal. De là, vous pouvez choisir de jouer depuis un plateau pré-enregistré (1), d'en générer un automatiquement (2), ou de quitter le jeu.

  1. Entrez le nom du fichier depuis lequel vous voulez jouer. Si vous voulez rajoutez vos propres fichiers, faites le en .csv, en utilisant un ";" comm séparateur, et enregistrez les dans le dossier "Resources" à la racine du projet.
  2. Pour générer un fichier aléatoirement selon vos propres règles, vous pouver modifier le fichier "Lettre.txt" du dossier "Resources". Celui ci est organisé de la manière suivante:
    • Le premier caractère est la lettre. le modifier entraînera un erreur
    • Le second paramètre est le nombre de fois maximal d'occurrences de la lettre sur le plateau
    • Le troisième paramètre est le score associé à la lettre, un peur comme au Scrabble.

Règles du jeu

Chaque joueur a un temps défini pour jouer. Passé ce délai le tour du prochain joueur commencera. Si vous entrez un mot ne se trouvant pas dans le dictionnaire français ou sur le plateau de jeu, et votre tour sera passé.
